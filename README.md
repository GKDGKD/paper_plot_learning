## jupyter notebook 转 markdown
1. 安装`nbconvert`包：
    ```pip install nbconvert```
2. 命令行转换：
    ```
    jupyter nbconvert --to markdown notebook.ipynb
    ```
    或者：
    ```
    jupyter nbconvert notebook.ipynb --to markdown
    ```
